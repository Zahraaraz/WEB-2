3. Git init adalah perintah yang di gunakan untuk membuat repositori git baru di direktori kerja lokal. perintah git init ini akan membuat direktori git yang berfungsi sbgai database untuk menyimpan info versioning.

   Git clone adalah perintah untuk meng-clone(menyalin) repositori git dri server remote /direktori lokal lainnya.

   Git add adalah perintah yang di gunakan untuk menambahkan perubahan pda file / direktori ke area staging.

   Git commit adalah perintah untuk menyimpan perubahan yang telah ditambahkan ke staging area ke dalam repositori git.

   Git push adalah perintah untuk mengirim perubahan yang telah di commit ke repositori di server.

   Git checkout adalah perintah untuk menukar cabang aktif dengan cabang yang dipilih.

   Git remote adalah perintah untuk mengelola daftar repositori jarak jauh yang terhubung dengan repositori lokal.
